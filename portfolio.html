Parler Farmer

Spring 2021

Working with three other computer science students, I participated in refactoring a Twitter Farmer to instead pull and store posts from the social media website Parler. I used Docker and MongoDB to store the posts, while the rest of the work was split up between my peers. The architecture of the project was designed to move Parler posts into a queue, while the backend would process the individual posts - meaning the frontend could work continuously.

Airline Hydraulics Webpage

Fall 2021 - Spring 2022

In the Spring of 2022, myself and three other students designed a webpage for the Airline Hydraulics Corporation. The problem we solved was to create an interface for administrators to easily modify employees in the existing database. Utilizing Bootstrap, React, and .NET, we successfully created a webpage which allowed for basic CRUD operations with a SQL database.

Sakila Movie Rental

Spring 2022

As a final project for my web development class at Wilkes University, I utilized a LAMP stack to create a functional movie rental website using the Sakila dataset. The frontend of the website was designed using Bootstrap, and the backend used PHP to communicate with a SQL database. Functionality included the creation, modification, reading, and deletion of movies in the database.
